Introduction to HTML and CSS

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

CSS (Cascading Style Sheets) is a style sheet language used for describing the look and formatting of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

Why use HTML?

HTML is pivotal to web development, providing the structure of web pages that users navigate on the internet. Without HTML, the web would not have its foundational structure.

Benefits of CSS

CSS offers web designers a solution to maintain consistency across different screen sizes, devices, and browsers. It also allows for a separation of content from design, enabling a more streamlined and efficient web design process.

Week 5 to 6: JavaScript

During these weeks, we explored JavaScript, a vital programming language for web development. JavaScript enables the creation of dynamically updating content, control multimedia, animate images, and much more, making web pages interactive and functional.

Week 7: DOM Events and Local Storage

This week focused on using DOM Events and Local Storage, essential for adding interactivity to web pages and storing data locally in the user's browser.

  • DOM Events: Learned to use events to trigger JavaScript functions, enhancing user interaction.
  • Local Storage: Gained knowledge on storing and retrieving data from the user's browser, allowing for persistent state between sessions.

Week 8: API from restdb

APIs are crucial for modern web development. This week, we learned about using restdb.io to integrate external databases into our projects seamlessly.

  • API Integration: Practiced making RESTful requests to restdb.io for storing and retrieving data.
  • Data Manipulation: Learned how to manipulate and display data fetched from an API in our web pages.

Week 9: Bootstrap

Bootstrap, a powerful front-end framework, was our focus this week. It's designed for faster and easier web development with responsive design in mind.

  • Responsive Design: Learned how to make web pages look good on all devices with Bootstrap's grid system and pre-designed components.
  • Bootstrap Components: Gained experience in utilizing Bootstrap's components for navigation, modals, and alerts to enhance user interfaces.

Week 9: Bootstrap

This week was all about Bootstrap, a front-end framework that enables developers to design beautiful, responsive websites quickly and efficiently. We covered the essentials of Bootstrap, including its grid system, components, and utilities, which are crucial for creating mobile-first websites.

  • Responsive Design: Mastered the grid system to create layouts that work on any device size.

Week 12: REST db

In week 12, our focus shifted towards understanding and using REST db, a cloud database service that uses REST API for database operations. This allowed us to explore how web applications can interact with databases in a more streamlined and efficient manner.

  • Database Integration: Learned to perform CRUD operations (Create, Read, Update, Delete) using RESTful APIs.
  • Data Modeling: Gained insights into structuring database schemas for optimal data storage and retrieval.

Week 13: Good Practices for Web Design

Week 13 was dedicated to learning about good practices in web design. This encompasses a wide range of best practices, including user interface design, user experience principles, accessibility, and web performance optimization.

  • User Experience (UX): Focused on designing with the user in mind to create intuitive and enjoyable web experiences.
  • Accessibility: Emphasized the importance of making web content accessible to users with disabilities, following WCAG guidelines.
  • Performance Optimization: Learned techniques for optimizing website speed and efficiency, enhancing user satisfaction.

Week 14: LOTTIE and Animation

The final week introduced us to LOTTIE, a library for adding lightweight, scalable animations to web applications. LOTTIE animations are based on Adobe After Effects, converted into a JSON format that can be played on any device without losing quality.

  • Integrating LOTTIE Animations: Learned how to incorporate LOTTIE animations into web projects, enhancing visual appeal and engagement.
  • Animation Principles: Explored the basics of animation, including timing, easing, and storytelling, to create meaningful interactions.
